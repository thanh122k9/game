# TikTok Live Stickman Game 🎮

Trò chơi đấu trường Stickman tương tác trực tiếp với TikTok Live.

## 🚀 Hướng dẫn khởi động nhanh

1. **Chạy file `run.bat`**: Double click vào file `run.bat` ở thư mục gốc.
   - Script sẽ tự động kiểm tra và cài đặt thư viện nếu cần (`npm install`).
   - Sau đó nó sẽ khởi động server tại `http://localhost:3000`.

2. **Mở trình duyệt**:
   - Truy cập `http://localhost:3000` để xem game.
   - Nhấn vào biểu tượng **Bánh răng ⚙️** (góc trên bên phải) để mở trang cài đặt.

3. **Kết nối TikTok**:
   - Trong trang cài đặt, nhập **ID TikTok** của bạn (ví dụ: `@username`).
   - Nhấn **THIẾT LẬP KẾT NỐI**.
   - Khi hiện "Đã kết nối thành công", game sẽ bắt đầu nhận lệnh từ livestream.

## 🕹️ Lệnh Chat (Mặc định)

- `!choi`: Tham gia trò chơi.
- `a`: Di chuyển sang trái.
- `d`: Di chuyển sang phải.
- `w`: Nhảy lên.
- `f`: Tấn công (Đấm/Súng/Kiếm).

## 🎁 Tương tác Quà tặng

- **Like**: Tự động kích hoạt tấn công.
- **Tặng quà (Gifts)**: Nhận ngẫu nhiên các Buff (Tăng công, Hồi máu, Giáp).

## 🛠️ Tùy chỉnh
Bạn có thể thay đổi sát thương vũ khí và phím lệnh trong trang `Cài đặt hệ thống`.
